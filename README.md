## Why BearScript Exists

Most programming languages are created for:
- Building websites and apps (JavaScript, Python)
- Creating systems software (C, Rust)
- Data science and AI (Python, R)

**BearScript is different.** It's created for:
- Making your **first lines of code** unforgettable
- Understanding **how programming languages actually work**
- Building **real programming skills** through playful metaphors

## Quick Start

```bash
# Clone and build
git clone https://github.com/rahmatadistiapro/BearScript
cd BearScript/bearscript
gcc -o bearscript [your source files]

# Start the interactive interpreter
./bearscript
>>> growl "Hello, Forest!"
Hello, Forest!

# Run a BearScript file
./bearscript examples/main.bearscript

```
